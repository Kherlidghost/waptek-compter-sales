# Role Dashboard Guide

This guide explains what each CompuMarket NG dashboard shows and why it is scoped that way.

## Admin Dashboard

The admin dashboard is the full marketplace control room. Admin users can see all branches, vendors, products, orders, payment receipts, repair requests, inventory, analytics, and sales reports.

Key views:

- Total marketplace orders, pending payments, confirmed payments, products, vendors, branches, and repair activity.
- Recent orders and recent payment receipts across all branches.
- Vendor approval queue for approving or rejecting sellers.
- Branch overview and sales reports.

Admin users are not restricted to one branch.

## Manager Dashboard

The manager dashboard is branch-specific. A manager should only see data for the branch assigned to their profile.

Key views:

- Assigned branch name and branch order totals.
- Branch pending payments and confirmed payments.
- Branch products, low-stock items, and out-of-stock items.
- Branch repair requests and branch sales overview.
- Recent branch orders and payment receipt activity.

Managers should not see other branch data or global admin settings.

## Cashier Dashboard

The cashier dashboard is focused on payment confirmation for the cashier’s assigned branch.

Key views:

- Pending receipts, confirmed payments, rejected receipts, and orders awaiting confirmation.
- Searchable receipt queue by order number, customer, or branch.
- Receipt image preview/open action.
- Confirm payment and reject receipt actions.
- Order detail access for relevant branch orders.

Cashiers cannot approve vendors, manage products, manage users, or view global analytics.

## Vendor Dashboard

The vendor dashboard shows only the vendor’s own marketplace activity.

Key views:

- Total products listed, active products, low-stock products, and out-of-stock products.
- Orders containing the vendor’s own products.
- Pending orders and paid/confirmed orders.
- Recent product activity and product performance summary.
- Product add/edit and inventory update actions.

Vendors cannot view other vendors’ products, other vendors’ orders, payment confirmation tools, or company-wide analytics.

## Customer Account Area

The customer account area helps buyers track their own marketplace activity.

Key views:

- Own orders and current order status.
- Pending/processing order count and paid/confirmed order count.
- Links to wishlist, repair request, product reviews, and continued shopping.
- Order detail links for uploaded receipt and payment confirmation status.

Customers can browse and add products to cart without logging in, but checkout/payment receipt upload and private account pages require authentication.
